## Sobre este projeto

Este projeto faz parte do meu estudo e prática no **Bootcamp Bradesco - GenAI & Dados**, realizado em parceria com a [DIO](https://www.dio.me/bootcamp/bradesco-genai-dados).

O objetivo é aplicar conceitos de **educação financeira** e **inteligência artificial** utilizando o **NotebookLM da Google**, organizando resumos, glossário e prompts reutilizáveis para apoiar revisões futuras.


## Bootcamp Bradesco - GenAI & Dados

Este bootcamp tem como foco o desenvolvimento de habilidades em **IA generativa** e **ciência de dados**, preparando profissionais para aplicar tecnologia em cenários reais de negócios e finanças.


## Objetivo do projeto
Organizar um guia de estudos em finanças pessoais utilizando o NotebookLM para resumir, estruturar e consolidar conhecimento a partir de múltiplas fontes confiáveis.

## Fontes utilizadas
- Vídeos: audiolivros, aulas e palestras sobre educação financeira e orçamento pessoal.
- PDFs: ebooks e relatórios de instituições financeiras.

## Estrutura do guia
- **resumo**: resumos estruturados dos conteúdos principais.
- **glossário**: conceitos financeiros explicados em linguagem simples.
- **prompts**: lista de prompts reutilizáveis para análise e estudo.

## Como usar o repositório
1. Consulte o NotebookLM (Guia Financeiro Digital): [Guia Financeiro Digital](https://notebooklm.google.com/notebook/3369d7d7-c335-4fe8-9bf9-fceaab258354)
2. Leia os resumos e o glossário na aba do Studio.
3. Use os prompts sugeridos no guia de estudo para novas análises e revisões.
4. Explore os recursos complementares (vídeos, podcasts, apresentações, mapas mentais e outros materiais) para aprofundar o estudo.
5. Faça perguntas, teste variações de prompts e utilize a inteligência artificial como apoio ativo para consolidar o conhecimento e esclarecer dúvidas.
