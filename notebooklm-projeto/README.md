# Guia de Estudo em Finanças Pessoais com IA (NotebookLM)

## Objetivo do projeto
Organizar um guia de estudos em finanças pessoais utilizando o NotebookLM para resumir, estruturar e consolidar conhecimento a partir de múltiplas fontes confiáveis.

## Fontes utilizadas
- Vídeos: aulas e palestras sobre educação financeira e orçamento pessoal.
- PDFs: ebooks e relatórios de instituições financeiras.
- Relatórios: documentos públicos com indicadores e análises setoriais.

## Estrutura do guia
- **resumo**: resumos estruturados dos conteúdos principais.
- **glossário**: conceitos financeiros explicados em linguagem simples.
- **prompts**: lista de prompts reutilizáveis para análise e estudo.

## Como usar o repositório
1. Consulte as fontes em notebooklm-projeto/fontes.
2. Leia os resumos e o glossário em notebooklm-projeto/guia-de-estudo.
3. Reaproveite os prompts para novas análises e revisões.
